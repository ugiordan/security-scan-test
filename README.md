# Security Scan Test Repository

This repository is used to test the security plugin system with GitHub Actions.

## Purpose

- Test all built-in security scanners (Gitleaks, Semgrep, etc.)
- Test external plugin integration (FIPS Compliance Checker)
- Verify end-to-end GitHub Actions workflow
- Validate finding aggregation and reporting

## Sample Code

This repository contains sample code with intentional security issues to verify scanners work correctly.
